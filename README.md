# Trading Setup App

A mobile-first trading discipline tool built as a single HTML file.  
Designed for crypto traders who want structure, consistency and professional habits.

## Features

- **Morning checklist** — news, BTC chart, mental state — unlocks the main screen only after completion
- **OLEGator setup** — trend-following strategy based on Bill Williams' Alligator + Fractals
- **RSI 30m setup** — counter-trend strategy with multi-timeframe RSI confirmation
- **Risk management tracker** — daily trade counter, loss limit bar, stop-day overlay at −10%
- **Mood filter** — blocks trading if emotional state is negative
- **Mentor report** — auto-generates a structured message for your mentor with one tap
- **Stop-day overlay** — full screen block when daily loss limit is reached

## How to Use

1. Download `index.html`
2. Open in any browser (Safari, Chrome, Edge)
3. On iPhone: open via Files app → Add to Home Screen
4. Use as a pre-trade ritual before every session

## Strategies

### 🐊 OLEGator (Trend)
- 10 candles passed after jaw closure
- Alligator mouth open and eating
- Fractal below jaw formed and broken
- Stop behind fractal above jaw

### 📉 RSI 30m (Counter-trend)
- RSI 30m + 1h above 70 or below 30
- Price at strong S/R level or trend line
- Reversal candle/formation present
- Stop and targets set via Fibonacci grid

## Risk Management

- 5% risk per trade
- Max 3 trades per day  
- Daily loss limit: −10%
- Auto stop-day lock when limit reached

## Tech

Pure HTML / CSS / JavaScript · No dependencies · Works fully offline
